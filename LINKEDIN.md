Use this for the project description on your LinkedIn profile:

Built a small home IT lab in VirtualBox using Windows 10 and Ubuntu 22.04 virtual machines. Practiced local user administration, Linux sudo permissions, VM snapshots, recovery mode, and basic network troubleshooting with NAT and Bridged Adapter configurations.

One issue I documented was losing expected sudo access after restoring an Ubuntu snapshot. I recovered the system through advanced/recovery mode, used a root shell, and restored sudo privileges to a local user. I also troubleshot VM connectivity by reviewing the VirtualBox network mode and switching from NAT to Bridged Adapter so the systems could communicate on the same network.

Skills practiced: VirtualBox, Windows 10 administration, Ubuntu Linux, user permissions, snapshots, recovery mode, IP connectivity, NAT, bridged networking, and troubleshooting documentation.
